Instrukcje testowania:

  1. Test pierwszego odwiedzenia:
    • Kliknij "Clear All Data"
    • Odśwież stronę
    • Powinien pojawić się popup z wyborem języka
  2. Test wyboru języka:
    • Wybierz jeden z 4 języków (Polski, English, Ελληνικά, עברית)
    • Popup powinien zniknąć
    • Status powinien się zaktualizować
  3. Test kolejnych wizyt:
    • Odśwież stronę ponownie
    • Popup NIE powinien się pojawić
    • Język powinien być zapamiętany

Oczekiwane zachowanie:

  • ✅ Popup pojawia się tylko przy pierwszym odwiedzeniu
  • ✅ Nie można zamknąć popupu bez wyboru (ESC disabled)
  • ✅ Nawigacja klawiaturą działa (Arrow Up/Down, Tab)
  • ✅ Po wyborze język jest zapisany w localStorage
  • ✅ Przy kolejnych wizytach popup się nie pokazuje
